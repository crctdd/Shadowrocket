使用方法如下：
1、复制这段链接‘https://raw.githubusercontent.com/crctdd/Shadowrocket-Auto/main/Shadowrocket-Auto.conf’
2、打开Shadowrocket，点击‘配置’-点击右上角‘+’，粘贴链接 点下载。

下载完点一下文件，弹窗点击‘使用配置’
